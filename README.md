# Selenium Tests for Wikipedia

Этот проект содержит автотесты на Python с использованием Selenium, которые проверяют:

1. **Позитивный тест** — что в атмосфере Земли по-прежнему содержится 20,95% кислорода.
2. **Негативный тест** — что при поиске несуществующей статьи Википедия корректно сообщает об отсутствии страницы.
